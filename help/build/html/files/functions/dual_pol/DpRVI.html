Dual-pol Radar Vegetation Index [MKR+20].

\[\text{DpRVI} = 1- \text{DOP}_{dp}\Big(\frac{\lambda_1}{\lambda_1+\lambda_2}\Big)\]

where,

\[\text{DOP}_{dp} = \sqrt{1-\frac{4\times \text{det ([C2])}}{\text{(Trace [C2])}^2}}\]

\(\text{[C2]}\) is co-variance matrix, and \(\lambda_1, \lambda_2\) are the eigen values of \(\langle\mathbf{[C2]}\rangle\) matrix in descending order.
